# Mediapipe RAM test
This is a test project used to demonstrate the mediapipe library's RAM increase even when the detectors are closed.

## How to deploy dockerfile
Run **build_docker.sh**
<br>
After that, execute:
```sh
   curl http://0.0.0.0:5100/process_frame/100
```
